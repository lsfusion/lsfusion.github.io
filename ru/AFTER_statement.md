# Инструкция AFTER

Инструкция `AFTER` - вызов [действия](/ru/Actions/.md) после вызова другого действия.

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
AFTER action(param1, ..., paramN) DO aspectAction;
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Инструкция `AFTER` задает действие (будем называть его *аспектом*), которое будет вызываться после вызова указанного действия.

### Параметры[​](#параметры "Прямая ссылка на этот заголовок")

* `action`

  [Идентификатор действия](/ru/IDs/.md#propertyid), после которого будет вызываться аспект.

* `param1, ..., paramN`

  Список имен параметров действия. Каждое имя задается [простым идентификатором](/ru/IDs/.md#id). К этим параметрам можно обращаться при задании аспекта.

* `aspectAction`

  [Контекстно-зависимый оператор-действие](/ru/Action_operators/.md#contextdependent), описывающий аспект.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
changePrice(Sku s, DATE d, NUMERIC[10,2] price)  { price(s, d) <- price; }
// Сообщение будет показано после каждого вызова changePrice
AFTER changePrice(Sku s, DATE d, NUMERIC[10,2] price) DO MESSAGE 'Price was changed'; 
```
