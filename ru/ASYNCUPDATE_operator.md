# Оператор ASYNCUPDATE

Оператор `ASYNCUPDATE` создает [действие](/ru/Actions/.md), реализующее [асинхронное обновление](/ru/State_change/.md#asyncupdate).

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
ASYNCUPDATE expr
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Оператор `ASYNCUPDATE` создает действие, которое вычисляет `expr` и отправляет полученное значение в открытый редактор на клиенте.

### Параметры[​](#параметры "Прямая ссылка на этот заголовок")

* `expr`

  [Выражение](/ru/Expression/.md), значение которого отправляется в открытый редактор.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
// возвращаем новое значение отображаемого кода в открытый редактор
onChangeSizeCode(Store store)  {
    DIALOG SelectStoreSize OBJECTS ss INPUT DO {
        storeSize(store) <- ss;
    }
    ASYNCUPDATE storeSizeCode(store);
}
```
