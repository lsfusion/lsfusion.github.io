# Установить автоматически

Автоматическая установка устанавливает платформу lsFusion вместе со всеми дополнительными приложениями, необходимыми для её работы (среда выполнения Java, PostgreSQL, сервер веб-приложений, среда разработки), при помощи одной программы установки или одного скрипта.
