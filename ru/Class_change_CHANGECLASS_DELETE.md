# Изменение класса (CHANGECLASS, DELETE)

Оператор *изменения класса* создает [действие](/ru/Actions/.md), которое устанавливает заданный [класс](/ru/Classes/.md) всем объектам, значение некоторого [свойства](/ru/Properties/.md) (*условия*) которых не равно `NULL`. Условие можно не задавать, в этом случае оно считается равным `TRUE`.

к сведению

Также в платформе существует встроенное действие `changeClass` с двумя параметрами: первый определяет объект, для которого необходимо изменить класс, а второй определяет объект нового класса. Так как при использовании встроенного действия определить возможные значения нового класса значительно сложнее, чем в случае с оператором (для которого новый класс задается явно), рекомендуется использовать именно оператор (а не встроенное действие)

Если существуют не `NULL` значения некоторых [первичных свойств](/ru/Data_properties_DATA/.md), для которых "изменяемый" объект присутствует в параметрах или самом значении и не подходит по классам, такие значения автоматически изменяются на `NULL`.

к сведению

Реализуется такое поведение по аналогии с [вычисляемыми](/ru/Calculated_events/.md) и [простыми](/ru/Simple_event/.md) событиями.

### Язык[​](#язык "Прямая ссылка на этот заголовок")

Для объявления действия, реализующего изменение классов объектов, используется [оператор `CHANGECLASS`](/ru/CHANGECLASS_operator/.md) или [оператор `DELETE`](/ru/DELETE_operator/.md).

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
// удаление объекта
deleteObject(obj)  { DELETE obj; }

// удаление всех неактивных товаров
CLASS Article;
active = DATA BOOLEAN (Article);
deleteInactiveArticles()  {
    // добавляется локальный параметр a, соответствующий перебираемым объектам
    DELETE Article a WHERE a IS Article AND NOT active(a); 
}
```

```
CLASS Document;
date = DATA DATE (Document);

CLASS ClosedDocument : Document;
// переводит все документы с датой старше 2-х недель в статус закрыт
changeStatus()  {
    CHANGECLASS Document d TO ClosedDocument WHERE sum(date(d), 14) <= currentDate();
}
```
