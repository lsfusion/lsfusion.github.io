# Оператор ROUND

Оператор `ROUND` - создание [свойства](/ru/Properties/.md), реализующего [операцию округления](/ru/Rounding_operator_ROUND/.md).

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
ROUND(numExpr[, scaleExpr])
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Оператор `ROUND` создает свойство, значением которого является число, округлённое до заданной точности. Описание класса результата и поведения округления, в том числе того, как точность определяет направление округления, см. в [операции округления](/ru/Rounding_operator_ROUND/.md).

### Параметры[​](#параметры "Прямая ссылка на этот заголовок")

* `numExpr`

  [Выражение](/ru/Expression/.md), значение которого определяет округляемое число. Значение должно принадлежать одному из числовых [встроенных классов](/ru/Built-in_classes/.md).

* `scaleExpr`

  Выражение, значение которого определяет количество разрядов, до которых округляется число. Значение должно принадлежать классу `INTEGER`. Если не указывается, число округляется до ближайшего целого.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
number = DATA NUMERIC[10,3]();  //12345.678
rounded = ROUND(number());      //12346
rounded1 = ROUND(number(), 2);  //12345.68
rounded2 = ROUND(number(), -2); //12300

FORM roundTest
PROPERTIES() number, rounded, rounded1, rounded2;
```
