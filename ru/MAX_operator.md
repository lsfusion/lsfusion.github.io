# Оператор MAX

Оператор `MAX` - создание [свойства](/ru/Properties/.md), реализующего нахождение [максимального](/ru/Extremum_MAX_MIN/.md) значения.

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
MAX expr1, ..., exprN
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Оператор `MAX` создает свойство, которое возвращает максимальное значение среди значений заданных свойств.

### Параметры[​](#параметры "Прямая ссылка на этот заголовок")

* `expr1, ..., exprN`

  Список [выражений](/ru/Expression/.md), среди значений которых выбирается максимальное.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
date1 = DATA DATE(INTEGER);
date2 = DATA DATE(INTEGER);
maxDate (INTEGER i) = MAX date1(i), date2(i);

balance = DATA INTEGER (Item);
outcome 'Остаток (неотрицательный)' (Item i) = MAX balance(i), 0;
```
