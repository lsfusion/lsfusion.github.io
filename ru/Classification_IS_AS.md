# Классификация (IS/AS)

Операторы *классификации* создают [свойства](/ru/Properties/.md), которые определяют принадлежность объекта заданному [классу](/ru/Classes/.md), который может быть [пользовательским классом](/ru/User_classes/.md) или [встроенным классом](/ru/Built-in_classes/.md). Значение создаваемого свойства зависит от того, принадлежит ли объект этому классу:

| Оператор | Объект принадлежит классу                   | Объект не принадлежит |
| -------- | ------------------------------------------- | --------------------- |
| `IS`     | логическое значение `TRUE`                  | `NULL`                |
| `AS`     | сам объект, принадлежащий указанному классу | `NULL`                |

### Язык[​](#язык "Прямая ссылка на этот заголовок")

Для реализации операторов классификации используется [операторы `IS` и `AS`](/ru/IS_AS_operators/.md).

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
asOrder(object) = object AS Order;

person = DATA Human (Order);
isMale (Order o) = person(o) IS Male;
```
