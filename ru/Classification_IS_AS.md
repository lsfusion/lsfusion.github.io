# Классификация (IS/AS)

Операторы *классификации* создают [свойства](/ru/Properties/.md), которые определяют принадлежность объекта заданному [классу](/ru/Classes/.md). В случае, если аргумент свойства не принадлежит указанному в операторе классу, то свойство возвращает `NULL`. Иначе оператор `IS` возвращает `TRUE`, а оператор `AS` возвращает переданный в качестве аргумента объект.

### Язык[​](#язык "Прямая ссылка на этот заголовок")

Для реализации операторов классификации используется [операторы `IS` и `AS`](/ru/IS_AS_operators/.md).

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
asOrder(object) = object AS Order;

person = DATA Human (Order);
isMale (Order o) = person(o) IS Male;
```
