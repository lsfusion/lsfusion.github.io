# Инструкция GROUP

Инструкция `GROUP` - создание новой [группы свойств и действий](/ru/Groups_of_properties_and_actions/.md).

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
GROUP [NATIVE] name [caption] [EXTID extID] [: parentName];
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Инструкция `GROUP` объявляет новую группу свойств и действий и добавляет ее в текущий [модуль](/ru/Modules/.md).

Ключевое слово `NATIVE` используется в некоторых системных модулях. Оно предназначено для объявления отдельных системных групп свойств и действий, которые создаются до инициализации модулей.

### Параметры[​](#параметры "Прямая ссылка на этот заголовок")

* `name`

  Имя группы. [Простой идентификатор](/ru/IDs/.md#id). Имя должно быть уникально в пределах текущего [пространства имен](/ru/Naming/.md#namespace).

* `caption`

  Заголовок группы. [Строковый литерал](/ru/Literals/.md#strliteral). Если заголовок не задан, то заголовком группы будет являться ее имя.

* `EXTID extID`

  Указание имени, которое будет использоваться для [экспорта / импорта](/ru/Structured_view/.md#extid) этой группы свойств. Используется только в [структурированном](/ru/Structured_view/.md) представлении.

  * `extId`

    Строковый литерал.

* `parentName`

  Имя родительской группы. [Составной идентификатор](/ru/IDs/.md#cid). Если имя родительской группы не задано, то родительской группой становится группа `System.private`.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
GROUP base : root; // Заголовком этой группы будет 'base'
GROUP local 'Локальные свойства'; // Родительской группой local будет являться System.private
```
