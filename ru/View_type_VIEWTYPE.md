# Вид отображения (VIEWTYPE)

Оператор вида отображения создаёт [свойство](/ru/Properties/.md), значением которого является текущий *вид отображения*, в котором [группа объектов](/ru/Form_structure/.md#objects) показывается пользователю: в виде таблицы, сводной таблицы, карты, календаря или пользовательского представления.

### Язык[​](#язык "Прямая ссылка на этот заголовок")

Для объявления свойства, возвращающего текущий вид отображения группы объектов, используется [оператор `VIEWTYPE`](/ru/Object_group_operator/.md).

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
CLASS Store;
name = DATA STRING[100] (Store);

FORM stores
    OBJECTS s = Store
    PROPERTIES(s) name
;
isPivot 'Склады в виде сводной таблицы' () = [ VIEWTYPE stores.s]() == ListViewType.pivot;
```
