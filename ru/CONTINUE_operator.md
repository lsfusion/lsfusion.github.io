# Оператор CONTINUE

Оператор `CONTINUE` создает [действие](/ru/Actions/.md), реализующее переход [к следующей итерации цикла](/ru/Next_iteration_CONTINUE/.md).

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
CONTINUE
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Оператор `CONTINUE` создает действие, которое реализует переход к следующей итерации наиболее вложенного цикла.

### Пример[​](#пример "Прямая ссылка на этот заголовок")

```
testContinue ()  {
    FOR iterate(INTEGER i, 1, 5) DO {
        MESSAGE 'before';
        IF i == 3 THEN CONTINUE; // нет сообщения 'after' при i == 3
        MESSAGE 'after';
    }
}
```
