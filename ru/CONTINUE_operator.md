# Оператор CONTINUE

Оператор `CONTINUE` - создание [действия](/ru/Actions/.md), реализующего переход [к следующей итерации цикла](/ru/Next_iteration_CONTINUE/.md).

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
CONTINUE
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Оператор `CONTINUE` создает действие, которое реализует переход к следующей итерации цикла.

### Пример[​](#пример "Прямая ссылка на этот заголовок")

```
testContinue ()  {
    FOR iterate(INTEGER i, 1, 5) DO {
        MESSAGE 'before';
        IF i == 3 THEN CONTINUE; // no message 'after' for i == 3
        MESSAGE 'after';
    }
}
```
