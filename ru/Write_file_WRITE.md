# Запись файла (WRITE)

Оператор *записи файла*, создает [действие](/ru/Actions/.md), которое читает файл (либо JSON-значение, которое сериализуется в файл с расширением `.json`) из значения некоторого свойства и сохраняет его по заданному URL.

URL для записи задается строковым значением. Поддерживаются следующие типы URL: **FILE**, **FTP**, **FTPS**, **SFTP**.

### Язык[​](#язык "Прямая ссылка на этот заголовок")

Для объявления действия, выполняющего запись файла, используется [оператор `WRITE`](/ru/WRITE_operator/.md).

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
loadAndWrite ()  {
    INPUT f = FILE DO {
        WRITE f TO 'file:///home/user/loadedfile.csv' APPEND;
        WRITE CLIENT f TO '/home/user/loadedfile.txt';
        WRITE CLIENT DIALOG f TO 'loadedfile';
    }
}
```
