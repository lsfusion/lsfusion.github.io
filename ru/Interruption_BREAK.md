# Прерывание (BREAK)

*Оператор прерывания* создает [действие](/ru/Actions/.md), которое выходит из наиболее вложенного цикла ([обычного](/ru/Loop_FOR/.md) или [рекурсивного](/ru/Recursive_loop_WHILE/.md)), внутри которого находится это действие. Управление передается первому действию, следующему за циклом. Если созданное действие не находится внутри цикла, оно выходит из наиболее вложенного [вызова действия](/ru/Call_EXEC/.md), как и [оператор выхода](/ru/Exit_RETURN/.md).

### Язык[​](#язык "Прямая ссылка на этот заголовок")

Синтаксис оператора прерывания описывается [оператором `BREAK`](/ru/BREAK_operator/.md).

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
testBreak ()  {
    FOR iterate(INTEGER i, 1, 100) DO {
        IF i == 50 THEN BREAK; // дойдет только до 50
    }
}
```
