# Идентификаторы

*Идентификаторы* в языке **lsFusion** используются для именования или обращения к [элементам системы](/ru/Element_identification/.md), параметрам [свойств](/ru/Properties/.md) и [действий](/ru/Actions/.md), [статическим объектам](/ru/Static_objects/.md), свойствам и действиям на форме и другим сущностям.

### Простой идентификатор[​](#id "Прямая ссылка на этот заголовок")

*Простой идентификатор* является базовым компонентом для создания других идентификаторов и представляет собой последовательность латинских букв любого регистра `a-zA-Z`, цифр `0-9`, и знака подчеркивания `_`. Первым символом простого идентификатора обязательно должна быть буква. Имена, начинающиеся с подчеркивания, зарезервированы для внутренних системных имен.

Простые идентификаторы используется в основном в качестве имен элементов системы и имен параметров свойств и действий.

#### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
name
value_id13
bankAccount
```

### Составной идентификатор[​](#cid "Прямая ссылка на этот заголовок")

```
[namespace.]name
```

*Составной идентификатор* используется для обращения к элементу системы по имени с возможностью указания [пространства имен](/ru/Naming/.md#namespace). Он может состоять либо из одного простого идентификатора, либо из двух простых идентификаторов, разделенных точкой. Если необходимо указать пространство имен, сначала указывается имя пространства, затем точка, после чего следует имя элемента системы.

#### Примеры[​](#примеры-1 "Прямая ссылка на этот заголовок")

```
System.name
Sale.Document
name
```

### Идентификатор класса[​](#classid "Прямая ссылка на этот заголовок")

*Идентификатор класса* используется для обозначения [пользовательского](/ru/User_classes/.md) или [встроенного](/ru/Built-in_classes/.md) класса. Для пользовательского класса идентификатор будет представлен в виде составного идентификатора, а для встроенного класса используются специальные [ключевые слова](/ru/Built-in_classes/.md), определяющие встроенный класс.

#### Примеры[​](#примеры-2 "Прямая ссылка на этот заголовок")

```
System.Object
Barcode
INTEGER
STRING
```

### Идентификатор статического объекта[​](#staticobjectid "Прямая ссылка на этот заголовок")

```
[namespace.]className.objectName
```

*Идентификатор статического объекта* используется для обозначения [статического объекта](/ru/Static_objects/.md) класса. Он состоит из идентификатора пользовательского класса и простого идентификатора (имя статического объекта), разделенных точкой.

#### Примеры[​](#примеры-3 "Прямая ссылка на этот заголовок")

```
Direction.north
System.FormResult.ok
```

### Идентификатор свойства или действия[​](#propertyid "Прямая ссылка на этот заголовок")

*Идентификатор свойства (действия)* используется для обозначения свойства (действия). В простом случае он является составным идентификатором, указывающим имя свойства (действия) и возможно его пространство имен, как и для любого другого элемента системы. В общем случае к этому составному идентификатору добавляется описание сигнатуры свойства (действия), которая описывает классы параметров свойства (действия). Сигнатура указывается в виде списка идентификаторов классов, заключенного в квадратные скобки. Если класс параметра неизвестен или не важен для однозначного указания свойства (действия), его идентификатор в списке можно заменить на вопросительный знак `?`.

#### Примеры[​](#примеры-4 "Прямая ссылка на этот заголовок")

```
userRole
Security.userRole
userRole[System.User]
cross[Circle, Line]
quantity[Document, ?, Store]
```

### Идентификатор свойства или действия на форме[​](#formpropertyid "Прямая ссылка на этот заголовок")

```
[namespace.]formName.formPropertyName
```

*Идентификатор свойства или действия на форме* используется для обозначения свойства или действия, добавленного на форму. Он состоит из составного идентификатора, указывающего форму, и [имени свойства (действия) на форме](/ru/Properties_and_actions_block/.md#name), разделенных точкой.

#### Примеры[​](#примеры-5 "Прямая ссылка на этот заголовок")

```
barcodeSku.amount(b)
Item.items.name(i)
Consignment.dashboard.date
```

### Идентификатор группы объектов[​](#groupobjectid "Прямая ссылка на этот заголовок")

```
[namespace.]formName.groupObjectName
```

*Идентификатор группы объектов* используется для обозначения [группы объектов](/ru/Form_structure/.md) (или объекта) на форме. Он состоит из составного идентификатора, указывающего форму, и имени группы объектов (либо имени объекта), разделенных точкой.

#### Примеры[​](#примеры-6 "Прямая ссылка на этот заголовок")

```
storeArticle.s
Item.form.object
```

### Типизированный параметр[​](#paramid "Прямая ссылка на этот заголовок")

```
[classID] name
```

*Типизированный параметр* используется для обозначения [параметра свойства](/ru/Properties/.md) (действия). Он состоит из необязательного идентификатора класса параметра и простого идентификатора, обозначающего имя параметра.

Если синтаксическая конструкция использует типизированные параметры, они интерпретируются в контексте параметров этой конструкции, если в статье о конструкции не сказано иное. В одном таком контексте одинаковые имена обозначают один и тот же параметр. Если класс параметра в этом контексте уже определен, повторно он не указывается. Должна ли конкретная конструкция создавать новый локальный контекст параметров, разрешать вводить новые параметры или определять их неявно, описывается в статье о самой конструкции.

#### Примеры[​](#примеры-7 "Прямая ссылка на этот заголовок")

```
user
User user
System.User user
INTEGER count
```
