# How-to: Навигатор

## Пример 1[​](#пример-1 "Прямая ссылка на этот заголовок")

### Условие[​](#условие "Прямая ссылка на этот заголовок")

Есть формы со списком книг и категорий.

```
FORM categories 'Категории';
FORM books 'Книги';
```

Нужно добавить их в [навигатор](/ru/Navigator/.md) в новую папку `'Приложение'` снизу от основного тулбара.

### Решение[​](#решение "Прямая ссылка на этот заголовок")

```
NAVIGATOR {
    NEW FOLDER application 'Приложение' WINDOW toolbar FIRST {
        NEW categories;
        NEW books;
    }
}
```

Указав `WINDOW` для элемента `application`, мы указали, что все его потомки должны отображаться в системном [окне](/ru/Navigator_design/.md) `toolbar`. Это будет выглядеть следующим образом :

## Пример 2[​](#пример-2 "Прямая ссылка на этот заголовок")

### Условие[​](#условие-1 "Прямая ссылка на этот заголовок")

Аналогичен [**Примеру 1**](#%D0%BF%D1%80%D0%B8%D0%BC%D0%B5%D1%80-1)

Нужно сделать, чтобы эти же формы были в подпапке `'Справочники'`.

### Решение[​](#решение-1 "Прямая ссылка на этот заголовок")

```
NAVIGATOR {
    application {
        NEW FOLDER masterData 'Справочники' {
            NEW categories;
            NEW books;
        }
    }
}
```
