## 袩褉懈屑械褉 1[鈥媇(#锌褉懈屑械褉-1 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

### 校褋谢芯胁懈械[鈥媇(#褍褋谢芯胁懈械 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

袝褋褌褜 蟹邪泻邪蟹 胁屑械褋褌械 褋芯 褋褌褉芯泻邪屑懈.

```
CLASS Order '袟邪泻邪蟹';
CLASS OrderDetail '小褌褉芯泻邪 蟹邪泻邪蟹邪';

order '袟邪泻邪蟹' = DATA Order (OrderDetail) NONULL DELETE;
```

袧械芯斜褏芯写懈屑芯 锌褉芯薪褍屑械褉芯胁邪褌褜 褋褌褉芯泻懈 芯褌 1 胁 写邪薪薪芯屑 蟹邪泻邪蟹械 胁 锌芯褉褟写泻械 懈褏 写芯斜邪胁谢械薪懈褟.

### 袪械褕械薪懈械[鈥媇(#褉械褕械薪懈械 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

```
index '袧芯屑械褉 褋褌褉芯泻懈' (OrderDetail d) = PARTITION SUM 1 ORDER d BY order(d) CHARWIDTH 4;
```

袙 写邪薪薪芯屑 褋谢褍褔邪械, 褋芯褉褌懈褉褍械屑 锌芯 胁薪褍褌褉械薪薪械屑褍 懈写械薪褌懈褎懈泻邪褌芯褉褍 褋褌褉芯泻懈 蟹邪泻邪蟹邪, 褌邪泻 泻邪泻 谐邪褉邪薪褌懈褉褍械褌褋褟, 褔褌芯 芯薪 胁芯蟹褉邪褋褌邪械褌 锌褉懈 褋芯蟹写邪薪懈懈 薪芯胁褘褏 褋褌褉芯泻.

## 袩褉懈屑械褉 2[鈥媇(#锌褉懈屑械褉-2 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

### 校褋谢芯胁懈械[鈥媇(#褍褋谢芯胁懈械-1 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

袝褋褌褜 褋锌懈褋芯泻 蟹邪泻邪蟹芯胁 锌芯泻褍锌邪褌械谢械泄 褋 褍泻邪蟹邪薪薪芯泄 写邪褌芯泄.

```
date '袛邪褌邪' = DATA DATE (Order);

CLASS Customer '袩芯泻褍锌邪褌械谢褜';
customer '袩芯泻褍锌邪褌械谢褜' = DATA Customer (Order);
```

袧械芯斜褏芯写懈屑芯 写谢褟 蟹邪泻邪蟹邪 薪邪泄褌懈 写邪褌褍 锌褉械写褘写褍褖械谐芯 蟹邪泻邪蟹邪 褝褌芯谐芯 卸械 锌芯泻褍锌邪褌械谢褟.

### 袪械褕械薪懈械[鈥媇(#褉械褕械薪懈械-1 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

```
prevOrderDate '袩褉械写褘写褍褖懈泄 蟹邪泻邪蟹' (Order o) = PARTITION PREV date(o) ORDER date(o), o BY customer(o);
```

袣邪泻 懈 胁 褋谢褍褔邪械 褋 [How-to: `GROUP CONCAT`](/ru/How-to_GROUP_CONCAT/.md), 锌芯褉褟写芯泻 写芯谢卸械薪 斜褘褌褜 锌芯谢薪芯褋褌褜褞 写械褌械褉屑懈薪懈褉芯胁邪薪. 袩芯褝褌芯屑褍 胁 `ORDER` 锌芯褋谢械写薪懈屑 锌邪褉邪屑械褌褉芯屑 写芯斜邪胁谢褟械褌褋褟 褋邪屑 蟹邪泻邪蟹 (写械-褎邪泻褌芯, 械谐芯 胁薪褍褌褉械薪薪懈泄 懈写械薪褌懈褎懈泻邪褌芯褉).

## 袩褉懈屑械褉 3[鈥媇(#锌褉懈屑械褉-3 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

### 校褋谢芯胁懈械[鈥媇(#褍褋谢芯胁懈械-2 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

袝褋褌褜 褌械泻褍褖懈械 芯褋褌邪褌泻懈 泻薪懈谐 锌芯 锌邪褉褌懈褟屑 薪邪 褋泻谢邪写械.

```
CLASS Book '袣薪懈谐邪';

CLASS Batch '袩邪褉褌懈褟';
book '袣薪懈谐邪' = DATA Book (Batch);
date '袛邪褌邪 锌褉懈褏芯写邪' = DATA DATE (Batch);

CLASS Stock '小泻谢邪写';
// 袨褋褌邪褌芯泻 褋写械谢邪薪 锌械褉胁懈褔薪褘屑 写谢褟 锌褉懈屑械褉邪. 袨斜褘褔薪芯 褝褌芯 胁褘褔懈褋谢褟械屑芯械 褋胁芯泄褋褌胁芯.
currentBalance '袨褋褌邪褌芯泻' = DATA INTEGER (Batch, Stock); 
```

袧械芯斜褏芯写懈屑芯 写谢褟 蟹邪写邪薪薪芯谐芯 泻芯谢懈褔械褋褌胁邪 芯写薪芯泄 泻薪懈谐懈 褉邪褋锌懈褋邪褌褜 褝褌芯 泻芯谢懈褔械褋褌胁芯 锌芯 锌邪褉褌懈褟屑 锌芯 锌褉懈薪褑懈锌褍 FIFO.

### 袪械褕械薪懈械[鈥媇(#褉械褕械薪懈械-2 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

```
quantity = DATA LOCAL INTEGER (Book);

quantityFIFO '袣芯谢-胁芯 锌芯 锌邪褉褌懈懈 FIFO' (Batch b, Stock s) = PARTITION UNGROUP quantity
                                                                    LIMIT STRICT currentBalance(b, s)
                                                                    ORDER date(b), b
                                                                    BY book(b);
```

袩邪褉邪屑械褌褉 `STRICT` 芯斜芯蟹薪邪褔邪械褌, 褔褌芯 械褋谢懈 泻芯谢懈褔械褋褌胁芯 斜褍写械褌 斜芯谢褜褕械 芯褋褌邪褌泻邪 锌芯 胁褋械屑 锌邪褉褌懈褟屑, 褌芯 胁褋褟 褉邪蟹薪懈褑邪 斜褍写械褌 写芯斜邪胁谢械薪邪 泻 锌芯褋谢械写薪械泄 锌邪褉褌懈懈.
