# Оператор BREAK

Оператор `BREAK` создает [действие](/ru/Actions/.md), реализующее [прерывание цикла](/ru/Interruption_BREAK/.md).

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
BREAK
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Оператор `BREAK` создает действие, которое выходит из наиболее вложенного цикла.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
testBreak ()  {
    FOR iterate(INTEGER i, 1, 100) DO {
        IF i == 50 THEN BREAK; // дойдет только до 50
    }
}
```
