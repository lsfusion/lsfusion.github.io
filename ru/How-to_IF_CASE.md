## 袩褉懈屑械褉 1[鈥媇(#锌褉懈屑械褉-1 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

### 校褋谢芯胁懈械[鈥媇(#褍褋谢芯胁懈械 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

袝褋褌褜 褋锌懈褋芯泻 泻薪懈谐, 泻芯褌芯褉褘械 锌褉懈胁褟蟹邪薪褘 泻 蟹邪写邪薪薪褘屑 泻邪褌械谐芯褉懈褟屑. 孝邪泻卸械 写谢褟 泻邪卸写芯泄 泻薪懈谐懈 蟹邪写邪薪邪 褑械薪邪.

```
CLASS Book '袣薪懈谐邪';
name '袧邪懈屑械薪芯胁邪薪懈械' = DATA ISTRING[50] (Book);

CLASS Category '袣邪褌械谐芯褉懈褟' {
    novel '袪芯屑邪薪',
    thriller '孝褉懈谢谢械褉',
    fiction '肖邪薪褌邪褋褌懈泻邪'
}

category '袣邪褌械谐芯褉懈褟' = DATA Category (Book);
price '笑械薪邪' = DATA NUMERIC[14,2] (Book);
```

袧褍卸薪芯 褋芯蟹写邪褌褜 写械泄褋褌胁懈械, 泻芯褌芯褉芯械 褍褋褌邪薪芯胁懈褌 芯锌褉械写械谢械薪薪褍褞 褑械薪褍 薪邪 泻薪懈谐褍, 械褋谢懈 芯薪邪 芯褌薪芯褋懈褌褋褟 泻 蟹邪写邪薪薪芯泄 泻邪褌械谐芯褉懈懈, 懈 褎懈泻褋懈褉芯胁邪薪薪褍褞 褑械薪褍 胁 锌褉芯褌懈胁薪芯屑 褋谢褍褔邪械. 袝褋谢懈 泻邪褌械谐芯褉懈褟 薪械 胁褘斜褉邪薪邪, 褌芯 写芯谢卸薪芯 斜褘褌褜 胁褘写邪薪芯 褋芯芯斜褖械薪懈械 芯斜 芯褕懈斜泻械.

### 袪械褕械薪懈械[鈥媇(#褉械褕械薪懈械 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

```
setPriceIf '校褋褌邪薪芯胁懈褌褜 褑械薪褍' (Book b)  {
    IF NOT category(b) THEN
        MESSAGE '袧械 胁褘斜褉邪薪邪 泻邪褌械谐芯褉懈褟 写谢褟 泻薪懈谐懈';
    ELSE
        IF category(b) == Category.novel THEN
            price(b) <- 50.0;
        ELSE
            price(b) <- 100.0;
}
```

## 袩褉懈屑械褉 2[鈥媇(#锌褉懈屑械褉-2 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

### 校褋谢芯胁懈械[鈥媇(#褍褋谢芯胁懈械-1 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

袗薪邪谢芯谐懈褔薪芯 [**袩褉懈屑械褉褍 1**](#%D0%BF%D1%80%D0%B8%D0%BC%D0%B5%D1%80-1).

袧褍卸薪芯 褋芯蟹写邪褌褜 写械泄褋褌胁懈械, 泻芯褌芯褉芯械 褍褋褌邪薪邪胁谢懈胁邪械褌 芯写薪褍 懈蟹 褌褉械褏 褑械薪 胁 蟹邪胁懈褋懈屑芯褋褌懈 芯褌 褌褉械褏 泻邪褌械谐芯褉懈泄. 袙芯 胁褋械褏 芯褋褌邪谢褜薪褘褏 褋谢褍褔邪褟褏 褑械薪邪 写芯谢卸薪邪 褋褌邪褌褜 褉邪胁薪芯泄 薪褍谢褞.

### 袪械褕械薪懈械[鈥媇(#褉械褕械薪懈械-1 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

```
setPriceCase '校褋褌邪薪芯胁懈褌褜 褑械薪褍' (Book b)  {
    CASE
        WHEN category(b) == Category.novel THEN
            price(b) <- 50.0;
        WHEN category(b) == Category.thriller THEN
            price(b) <- 100.0;
        WHEN category(b) == Category.fiction THEN
            price(b) <- 150.0;
    ELSE
        price(b) <- 0.0;
}
```
