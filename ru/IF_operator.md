# Оператор IF

Оператор `IF` - создание [свойства](/ru/Properties/.md), реализующего [выбор](/ru/Selection_CASE_IF_MULTI_OVERRIDE_EXCLUSIVE/.md) значения по условию (одиночная форма).

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
result IF condition 
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Оператор `IF` создает свойство, которое возвращает заданное значение при выполнении некоторого условия. Если условие не выполняется, свойство возвращает `NULL`.

### Параметры[​](#параметры "Прямая ссылка на этот заголовок")

* `result`

  [Выражение](/ru/Expression/.md), значение которого определяет результат.

* `condition`

  Выражение, значение которого определяет условие.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
name = DATA STRING[100] (Book);
hasName (Book b) = TRUE IF name(b);

background (Book b) = RGB(224, 255, 128) IF b IS Book;

countTags (Book b) = GROUP SUM 1 IF in(b, Tag t);
```
