# Рекурсивный цикл (WHILE)

Оператор *рекурсивного цикла* аналогичен оператору обычного [цикла](/ru/Loop_FOR/.md), с той лишь разницей, что итерирование выполняется рекурсивно до того момента, пока на некотором шаге множество наборов объектов удовлетворяющих условию не станет пустым. Как следствие, для этого оператора нет смысла в задании *альтернативного* действия.

### Язык[​](#язык "Прямая ссылка на этот заголовок")

Синтаксис оператора рекурсивного цикла описывается [оператором `WHILE`](/ru/WHILE_operator/.md).

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
iterateDates (DATE dateFrom, DATE dateTo)  {
    LOCAL dateCur = DATE();

    dateCur() <- dateFrom;
    WHILE dateCur() <= dateTo DO {
        MESSAGE 'I have a date ' + dateCur();
        dateCur() <- sum(dateCur(), 1);
    }
}
```
