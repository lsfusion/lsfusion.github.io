# Оператор CLOSE FORM

Оператор `CLOSE FORM` - создание [действия](/ru/Actions/.md), которое закрывает открытый в [интерактивном представлении](/ru/In_an_interactive_view_SHOW_DIALOG/.md) экземпляр формы с заданным идентификатором.

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
CLOSE FORM formId
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Оператор `CLOSE FORM` создаёт действие, которое закрывает у пользователя открытый экземпляр формы с идентификатором `formId` (в виде запроса клиенту). Идентификатор задаётся при открытии формы [оператором `SHOW`](/ru/SHOW_operator/.md). Действие не имеет параметров и не использует [контекст](/ru/Action_operators/.md#contextdependent). Если открыто несколько экземпляров с этим идентификатором, закрываются все они; если ни одного такого экземпляра нет, действие не имеет эффекта.

### Параметры[​](#параметры "Прямая ссылка на этот заголовок")

* `formId`

  [Строковый литерал](/ru/Literals/.md#strliteral), идентифицирующий закрываемый экземпляр формы. Это тот же идентификатор, который задаётся при открытии формы оператором `SHOW`.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
FORM orders
    OBJECTS o = Order
    PROPERTIES(o) number, customer
;

// открытие формы с идентификатором экземпляра
openOrders ()  {
    SHOW 'ordersInstance' = orders NOWAIT;
}

// закрытие ранее открытого экземпляра по его идентификатору
closeOrders ()  {
    CLOSE FORM 'ordersInstance';
}
```
