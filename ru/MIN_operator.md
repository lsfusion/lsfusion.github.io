# Оператор MIN

Оператор `MIN` - создание [свойства](/ru/Properties/.md), реализующего нахождение [минимального](/ru/Extremum_MAX_MIN/.md) значения.

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
MIN expr1, ..., exprN
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Оператор `MIN` создает свойство, значением которого является минимальное среди значений заданных операндов. Пропуск `NULL`-операндов и определение класса результата соответствуют [экстремуму](/ru/Extremum_MAX_MIN/.md).

### Параметры[​](#параметры "Прямая ссылка на этот заголовок")

* `expr1, ..., exprN`

  [Выражения](/ru/Expression/.md), среди значений которых выбирается минимальное. Должен быть задан хотя бы один операнд.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
price1 = DATA NUMERIC[10,2] (Book);
price2 = DATA NUMERIC[10,2] (Book);
minPrice (Book b) = MIN price1(b), price2(b);

date = DATA DATE (INTEGER);
minDate (INTEGER i) = MIN date(i), 2001_01_01;
```
