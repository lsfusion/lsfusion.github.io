# Операции с классами

Операции с классами включают в себя следующие операторы:

* [Классификация (`IS`/`AS`)](/ru/Classification_IS_AS/.md) — определяют принадлежность объекта классу.
* [Принадлежность сигнатуре (`ISCLASS`)](/ru/Property_signature_ISCLASS/.md) — определяет, может ли свойство иметь значение для заданных объектов, исходя из классов этих объектов.
