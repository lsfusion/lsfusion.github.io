# Идентификация элементов

В платформе **lsFusion** некоторые элементы системы является *именованными*, это означает, что их можно использовать, обратившись к ним по имени.

Именованными элементами системы могут быть: [cвойства](/ru/Properties/.md), [действия](/ru/Actions/.md), [пользовательские классы](/ru/User_classes/.md), [формы](/ru/Forms/.md), [элементы навигатора](/ru/Navigator/.md), [группы свойств](/ru/Groups_of_properties_and_actions/.md), [окна](/ru/Navigator_design/.md), [таблицы](/ru/Tables/.md), [метакоды](/ru/Metaprogramming/.md#metacode).

*Идентификация* этих элементов осуществляется при помощи их [имени](/ru/Naming/.md), а также, в отдельных случаях, с учетом семантики этих элементов. Обращение к элементам системы происходит по правилам, описанным в статье [поиск](/ru/Search_/.md) элементов.
