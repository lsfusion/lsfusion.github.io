# How-to: Действия

### Порядок ознакомления[​](#порядок-ознакомления "Прямая ссылка на этот заголовок")

1. [`CHANGE`](/ru/How-to_CHANGE/.md). Запись значений в свойства.
2. [`EXEC`](/ru/How-to_EXEC/.md). Вызов одного действия из другого.
3. [`NEW`](/ru/How-to_NEW/.md). Создание нового объекта.
4. [`DELETE`](/ru/How-to_DELETE/.md). Удаление одного или нескольких объектов.
5. [`FOR`](/ru/How-to_FOR/.md). Операция цикла.
6. [`IF`/`CASE`](/ru/How-to_IF_CASE/.md). Ветвление выполнения.
7. [`WHILE`](/ru/How-to_WHILE/.md). Операция рекурсивного цикла.
8. [`NEWSESSION`](/ru/How-to_NEWSESSION/.md). Создание новой сессии изменений.
9. [`ACTIVATE`](/ru/How-to_ACTIVATE/.md). Изменение текущего объекта в форме.
