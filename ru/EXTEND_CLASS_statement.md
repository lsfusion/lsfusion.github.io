# Инструкция EXTEND CLASS

Инструкция `EXTEND CLASS` - [расширение](/ru/Class_extension/.md) существующего класса.

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
EXTEND CLASS name 
[{
    objectName1 [objectCaption1] [imageSetting],
    ...
    objectNameM [objectCaptionM] [imageSetting]
}] 
[: parent1, ..., parentN];
```

Где `imageSetting` принимает один из видов:

```
IMAGE [imageLiteral]
NOIMAGE
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Инструкция `EXTEND CLASS` расширяет существующий [пользовательский класс](/ru/User_classes/.md) дополнительными родительскими классами и новыми [статическими объектами](/ru/Static_objects/.md). Расширять можно, в том числе и [абстрактные классы](/ru/User_classes/.md#abstract), добавляя им родительские классы.

### Параметры[​](#параметры "Прямая ссылка на этот заголовок")

* `name`

  Имя класса. [Составной идентификатор](/ru/IDs/.md#cid).

* `objectName1, ..., objectNameM`

  Имена новых статических объектов указанного класса. Каждое имя задается [простым идентификатором](/ru/IDs/.md#id). Свойство `name[StaticObject]` возвращает это имя вместе с пространством имён и классом — [каноническое имя](/ru/Static_objects/.md) объекта.

* `objectCaption1, ..., objectCaptionM`

  Заголовки новых статических объектов указанного класса. Каждый заголовок является [строковым литералом](/ru/Literals/.md#strliteral). Если заголовок не задан, то заголовком статического объекта будет являться его имя. Заголовок каждого статического объекта доступен через свойство `caption[StaticObject]`.

* `imageSetting`

  Настройка иконки статического объекта. Один из вариантов:

  * `IMAGE`

    [Указание иконки](/ru/Icons/.md#manual) вручную, после которого может следовать `imageLiteral` — [строковый литерал](/ru/Literals/.md#strliteral), значение которого определяет иконку. Если `imageLiteral` не указан, происходит переключение в режим [автоматической установки](/ru/Icons/.md#auto) иконки.

  * `NOIMAGE`

    У статического объекта не должно быть иконки.

* `parent1, ..., parentN`

  Список имен новых родительских классов. Каждое имя задается составным идентификатором.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
CLASS ABSTRACT Shape;
CLASS Box : Shape;

CLASS Quadrilateral;
EXTEND CLASS Box : Quadrilateral; // Добавляем наследование

CLASS ShapeType {
	point 'Точка',
	segment 'Отрезок'
}

EXTEND CLASS ShapeType { // Добавляем статический объект
	circle 'Окружность'
}
```
