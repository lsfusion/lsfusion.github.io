# Оператор \[]

Оператор `[]` - создание [свойства](/ru/Properties/.md), возвращающего объект из [структуры](/ru/Structure_operators_STRUCT/.md).

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
expr[n]
```

Где, `[` и `]` - это обычные квадратные скобки.

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Оператор `[]` создает свойство, принимающее на вход структуру и возвращающее объект [структуры](/ru/Structure_operators_STRUCT/.md), стоящий на позиции, заданной числом `n`.

### Параметры[​](#параметры "Прямая ссылка на этот заголовок")

* `expr`

  [Выражение](/ru/Expression/.md), значением которого должна являться структура.

* `n`

  Позиция объекта внутри структуры. Положительный [числовой литерал](/ru/Literals/.md#intliteral): должен записываться как константа, а не вычисляться выражением. Нумерация начинается с 1, и значение должно быть в интервале `[1..N]`, где `N` - количество объектов в структуре.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
CLASS Letter;
attachment1 = DATA FILE (Letter);
attachment2 = DATA FILE (Letter);
letterAttachments (Letter l) = STRUCT(attachment1(l), attachment2(l));
secondAttachment(Letter l) = letterAttachments(l)[2]; // возвращает attachment2
```
