袟邪写邪薪褘 锌芯薪褟褌懈褟 泻薪懈谐 懈 锌芯泻褍锌邪褌械谢械泄. 袛谢褟 泻邪卸写芯泄 泻薪懈谐懈 懈 锌芯泻褍锌邪褌械谢褟 蟹邪写邪械褌褋褟 褋胁芯褟 褑械薪邪.

袧褍卸薪芯 锌芯褋褌褉芯懈褌褜 褎芯褉屑褍 褋 械写懈薪褋褌胁械薪薪芯泄 褌邪斜谢懈褑械泄, 胁 泻芯褌芯褉芯泄 胁 褋褌褉芯泻邪褏 斜褍写褍褌 泻薪懈谐懈, 邪 胁 泻芯谢芯薪泻邪褏 - 锌芯泻褍锌邪褌械谢懈. 袙 泻邪卸写芯泄 褟褔械泄泻械 写芯谢卸薪邪 斜褘褌褜 褑械薪邪 写谢褟 写邪薪薪芯泄 泻薪懈谐懈 懈 锌芯泻褍锌邪褌械谢褟.

### 袪械褕械薪懈械[鈥媇(#褉械褕械薪懈械 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

### 校褋谢芯胁懈械[鈥媇(#褍褋谢芯胁懈械-1 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

袗薪邪谢芯谐懈褔薪芯 [**袩褉懈屑械褉褍 1**](#%D0%BF%D1%80%D0%B8%D0%BC%D0%B5%D1%80-1), 褌芯谢褜泻芯 写谢褟 泻薪懈谐懈 懈 锌芯泻褍锌邪褌械谢褟 褌邪泻卸械 蟹邪写邪械褌褋褟 芯褌褋褉芯褔泻邪 锌谢邪褌械卸邪 胁 写薪褟褏.

```
gracePeriod '袨褌褋褉芯褔泻邪, 写薪械泄' = DATA INTEGER (Book, Customer);
```

袧褍卸薪芯 锌芯褋褌褉芯懈褌褜 褎芯褉屑褍 褋 械写懈薪褋褌胁械薪薪芯泄 褌邪斜谢懈褑械泄, 胁 泻芯褌芯褉芯泄 胁 褋褌褉芯泻邪褏 斜褍写褍褌 泻薪懈谐懈, 邪 胁 泻芯谢芯薪泻邪褏 - 锌芯泻褍锌邪褌械谢懈. 袩褉懈 褝褌芯屑 写谢褟 泻邪卸写芯谐芯 锌芯泻褍锌邪褌械谢褟 写芯谢卸薪芯 斜褘褌褜 写胁械 泻芯谢芯薪泻懈 褉褟写芯屑 : 褑械薪邪 懈 芯褌褋褉芯褔泻邪. 袣芯谢芯薪泻懈 褋 芯褌褋褉芯褔泻芯泄 薪褍卸薪芯 胁褘写械谢懈褌褜 卸械谢褌褘屑 褑胁械褌芯屑. 袛芯谢卸薪邪 斜褘褌褜 胁芯蟹屑芯卸薪芯褋褌褜 胁褘斜懈褉邪褌褜 泻邪泻懈褏 锌芯泻褍锌邪褌械谢械泄 芯褌芯斜褉邪卸邪褌褜 胁 泻芯谢芯薪泻懈.

### 袪械褕械薪懈械[鈥媇(#褉械褕械薪懈械-1 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

```
selected '袨褌屑' = DATA BOOLEAN (Customer);

headerName '袟邪谐芯谢芯胁芯泻 褑械薪褘' (Customer c) = name(c) + ': 笑械薪邪';
headerGrace '袟邪谐芯谢芯胁芯泻 芯褌褋褉芯褔泻懈' (Customer c) = name(c) + ': 袨褌褋褉芯褔泻邪, 写薪械泄';

FORM pricesAndGracePeriods '笑械薪褘 懈 芯褌褋褉芯褔泻懈'
    OBJECTS s = Customer
    PROPERTIES selected(s), name(s) READONLY

    OBJECTS c = Customer
    FILTERS selected(c)

    OBJECTS b = Book
    PROPERTIES name(b) READONLY,
               price(b, c) COLUMNS 'priceAndGrace' (c) HEADER headerName(c),
               gracePeriod(b, c) COLUMNS 'priceAndGrace' (c) HEADER headerGrace(c)
;

DESIGN pricesAndGracePeriods {
    BOX(b) {
        fill = 3;
        PROPERTY(gracePeriod(b, c)) {
            background = #FFFFAA;
        }
    }
}
```

袙 泻芯谢芯薪泻懈 斜褍写褍褌 锌芯泻邪蟹褘胁邪褌褜褋褟 褌芯谢褜泻芯 褌械 芯斜褗械泻褌褘, 泻芯褌芯褉褘械 锌芯锌邪写邪褞褌 胁 褎懈谢褜褌褉 写谢褟 芯斜褗械泻褌邪 `c`. 袣褉芯屑械 褌芯谐芯, 泻芯谢芯薪泻懈 斜褍写褍褌 懈写褌懈 胁 褌芯屑 卸械 锌芯褉褟写泻械, 褔褌芯 懈 芯斜褗械泻褌褘.

袙褋械 褋胁芯泄褋褌胁邪 褋 芯写懈薪邪泻芯胁褘屑 懈写械薪褌懈褎懈泻邪褌芯褉芯屑 锌芯褋谢械 褋谢芯胁邪 `COLUMNS` 斜褍写褍褌 懈写褌懈 褉褟写芯屑 写谢褟 芯写薪懈褏 懈 褌械褏 卸械 芯斜褗械泻褌芯胁. 袙 写邪薪薪芯屑 褋谢褍褔邪械, 褑械薪邪 懈 芯褌褋褉芯褔泻邪 写谢褟 锌械褉胁芯谐芯 锌芯泻褍锌邪褌械谢褟, 蟹邪褌械屑 褑械薪邪 懈 芯褌褋褉芯褔泻邪 写谢褟 胁褌芯褉芯谐芯 懈 褌邪泻 写邪谢械械. 袝褋谢懈 懈写械薪褌懈褎懈泻邪褌芯褉褘 斜褘谢懈 斜褘 褉邪蟹薪褘屑懈 懈谢懈 薪械 褍泻邪蟹邪薪褘 胁芯 胁褋械, 褌芯 褋薪邪褔邪谢邪 斜褘 褕谢懈 褑械薪褘 写谢褟 胁褋械褏 锌芯泻褍锌邪褌械谢械泄, 邪 锌芯褌芯屑 褍卸械 芯褌褋褉芯褔泻懈.

袧褍卸薪芯 锌芯褋褌褉芯懈褌褜 褎芯褉屑褍, 胁 泻芯褌芯褉芯泄 斜褍写械褌 屑邪褌褉懈褑邪 褋 蟹邪写邪胁邪械屑褘屑 泻芯谢懈褔械褋褌胁芯屑 褉褟写芯胁 懈 泻芯谢芯薪芯泻. 袩芯谢褜蟹芯胁邪褌械谢褜 写芯谢卸械薪 懈屑械褌褜 胁芯蟹屑芯卸薪芯褋褌褜 锌褉芯褋褌邪胁谢褟褌褜 胁 泻邪卸写芯泄 褟褔械泄泻械 芯褌屑械褌泻褍.

### 袪械褕械薪懈械[鈥媇(#褉械褕械薪懈械-2 "袩褉褟屑邪褟 褋褋褘谢泻邪 薪邪 褝褌芯褌 蟹邪谐芯谢芯胁芯泻")

```
in '袙泻谢' = DATA BOOLEAN (INTEGER, INTEGER);

row '袟邪谐芯谢芯胁芯泻 褉褟写邪' (INTEGER r) = '袪褟写 ' + (r AS INTEGER);
column '袟邪谐芯谢芯胁芯泻 泻芯谢芯薪泻懈' (INTEGER c) = '袣芯谢芯薪泻邪 ' + (c AS INTEGER);

FORM matrix '袦邪褌褉懈褑邪'
    OBJECTS m = (rows = INTEGER, columns = INTEGER) PANEL
    PROPERTIES rs = VALUE(rows), cs = VALUE(columns)

    OBJECTS c = INTEGER
    FILTERS iterate(c, 1, columns)

    OBJECTS r = INTEGER
    PROPERTIES(r) row
    PROPERTIES in(r, c) COLUMNS (c) HEADER column(c)
    FILTERS iterate(r, 1, rows)
;

DESIGN matrix {
    PROPERTY(rs) { caption = '袣芯谢-胁芯 褉褟写芯胁'; }
    PROPERTY(cs) { caption = '袣芯谢-胁芯 泻芯谢芯薪芯泻'; }
}
```
