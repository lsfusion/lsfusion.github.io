# Оператор CHANGECLASS

Оператор `CHANGECLASS` создает [действие](/ru/Actions/.md), которое [изменяет классы объектов](/ru/Class_change_CHANGECLASS_DELETE/.md).

### Синтаксис[​](#синтаксис "Прямая ссылка на этот заголовок")

```
CHANGECLASS expr TO className [WHERE whereExpr]
```

### Описание[​](#описание "Прямая ссылка на этот заголовок")

Оператор `CHANGECLASS` создает действие, которое присваивает класс `className` объекту, заданному выражением `expr`, для каждого набора аргументов, для которого `whereExpr` не равно `NULL`.

Оператор может вводить локальный параметр в `expr`; в этом случае блок `WHERE` обязателен. Такой параметр соответствует перебираемым объектам и не является параметром создаваемого действия.

### Параметры[​](#параметры "Прямая ссылка на этот заголовок")

* `expr`

  [Выражение](/ru/Expression/.md) или [типизированный параметр](/ru/IDs/.md#paramid) для объекта, класс которого изменяется. В качестве типизированного параметра можно использовать как уже объявленный параметр, так и объявить новый локальный параметр; при использовании выражения добавлять новые локальные параметры нельзя.

* `className`

  Имя [пользовательского класса](/ru/User_classes/.md), в который переводится класс объекта. [Составной идентификатор](/ru/IDs/.md#cid). Класс должен быть конкретным.

* `whereExpr`

  Выражение, значение которого является условием изменения класса. Если не задано, считается равным `TRUE`.

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
CLASS Document;
date = DATA DATE (Document);

CLASS ClosedDocument : Document;
// переводит все документы с датой старше 2-х недель в статус закрыт
changeStatus()  {
    CHANGECLASS Document d TO ClosedDocument WHERE sum(date(d), 14) <= currentDate();
}
```
