# Выход (RETURN)

*Оператор выхода* создает [действие](/ru/Actions/.md), которое выходит из наиболее вложенного [вызова действия](/ru/Call_EXEC/.md). Управление передается первому действию, следующему за этим оператором вызова.

Оператор выхода может также задавать [результат](/ru/Actions/.md) окружающего [вызова действия](/ru/Call_EXEC/.md).

### Язык[​](#язык "Прямая ссылка на этот заголовок")

Синтаксис оператора выхода описывается [оператором `RETURN`](/ru/RETURN_operator/.md).

### Примеры[​](#примеры "Прямая ссылка на этот заголовок")

```
// выход без значения
importFile  {
    LOCAL file = FILE ();
    INPUT f = FILE DO {
        file () <- f;
    }

    IF NOT file() THEN RETURN;
}

// выход со значением — значение становится результатом окружающего вызова действия
priceBucket (INTEGER price)  {
    IF price > 1000 THEN RETURN 'high';
    IF price > 100 THEN RETURN 'mid';
    RETURN 'low';
}
```
