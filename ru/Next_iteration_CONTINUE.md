# Следующая итерация (CONTINUE)

Оператор *перехода к следующей итерации* создает [действие](/ru/Actions/.md), которое пропускает выполнение оставшегося кода в текущей итерации и переходит на следующую итерацию цикла ([обычного](/ru/Loop_FOR/.md) или [рекурсивного](/ru/Recursive_loop_WHILE/.md)). Если созданное действие не находится внутри цикла, то его поведение становится аналогичным действию, созданному [оператором выхода](/ru/Exit_RETURN/.md).

### Язык[​](#язык "Прямая ссылка на этот заголовок")

Синтаксис оператора перехода к следующей итерации описывается [оператором `CONTINUE`](/ru/CONTINUE_operator/.md).

### Пример[​](#пример "Прямая ссылка на этот заголовок")

```
testContinue ()  {
    FOR iterate(INTEGER i, 1, 5) DO {
        MESSAGE 'before';
        IF i == 3 THEN CONTINUE; // no message 'after' for i == 3
        MESSAGE 'after';
}
```
